# Aquata

This is the remake of the Browsergame Aquata https://aquatation.de.

## Develop

### Prerequisites

- docker
- node

Create local database
```
./postgres.sh
```

Start backend
```shell script
npm run nx -- serve api
```

Start frontend
```shell script
npm run nx -- serve aquata
```
